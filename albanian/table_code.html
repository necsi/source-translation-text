Protokolli i reagimit Gjelber Verdh Portokalli Kuq
Vetëdijesimi i publikut, duke përfshirë mekanizmat e bartjes së sëmundjes dhe shmangies së saj (distancimi shoqëror, pastrimi i duarve, maskat) ✓ ✓ ✓ ✓
Zhvillono proces të reagimit të shpejt për të testuar qytetarët vendor për simptomat relevante. ✓ ✓ ✓ ✓
Bëni teste të shpejta të individëve në kufij që vijnë nga zonat e Verdha dhe Portokalli për të identifikuar individët me simptoma (ethe,kollë). Udhëtarët që hyjnë me të njëjtin mjet transporti (aeroplan, tren, autobus, automobilë) do të duhej të mbahen deri në marrjen e rezultatit. Nëse testojnë pozitiv, detyroni rastet e konfirmuara, si dhe udhëtarët tjera, në karantinë prej 14 ditëve (individualisht apo në grupe të vogëla) ✓ ✓ ✓ ✓
Personat që ka gjasë se kanë pas kontakt me individët e infektuar i detyroni në karantinim prej 14 ditësh, duke përfshirë udhëtarët nga Zonat e Kuqe. ✓ ✓ ✓ ✓
Identifikoni kontaktet e rasteve të konfirmuar (gjurmimi i kontakteve), dhe testoni, monitoroni, dhe/apo kërkoni vet-izolim nga ta.   ✓ ✓ ✓
Nxitni mbrojtjen personale duke përfshirë distancimin shoqëror, pastrimin e duardve, etiketën e teshtitjes   ✓ ✓ ✓
Monitoroni shpesh gjendjen shëndetësore të grupeve të veçanta që kanë kontakt të shpeshtë shoqëror, posaçërisht në zonat ku janë identifikuar raste kontakti në komunitet, që të keni detektim të hershëm të rasteve dhe shpërthimit   ✓ ✓ ✓
Nxitni qytetarët që t'i shmangen tubimeve që nuk janë të domosdoshëm, veçanërisht në hapësirat e mbyllura   ✓ ✓ ✓
Siguroni mbrojtje maksimale për stafin mjekësor   ✓ ✓ ✓
