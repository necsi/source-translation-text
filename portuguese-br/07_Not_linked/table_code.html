Aumentar a conscientização do público, incluindo mecanismos de transmissão de doenças e prevenção (distanciamento social, lavagem das mãos, máscaras) ✓ ✓ ✓ ✓
Desenvolver um processo de resposta rápida para testar os cidadãos locais quanto a sintomas relacionados ✓ ✓ ✓ ✓
Realizar testes diagnósticos rápidos nas fronteiras em indivíduos que chegam de viagem das Zonas Amarela ou Laranja, para identificar os indivíduos com sintomas (febre, tosse). Todos os passageiros que estão no mesmo veículo (avião, trem, ônibus, carro) devem ser retidos para o resultado. Se positivo, impor quarentena de 14 dias do caso confirmado e de outros passageiros e tripulantes (individualmente ou em pequenos grupos) ✓ ✓ ✓ ✓
Impor quarentena de 14 dias para indivíduos que provavelmente tiveram contato com indivíduos infectados, incluindo todos os viajantes que vieram das Zonas Vermelhas ✓ ✓ ✓ ✓
Identificar as pessoas que tiveram contato com portadores de casos confirmados (rastreamento de contatos) e testá-los, monitorá-los e/ou isolá-los   ✓ ✓ ✓
Reforçar hábitos de proteção pessoal, incluindo distanciamento social, lavagem das mãos, e formas corretas de conter espirros   ✓ ✓ ✓
